# Package index

## All functions

- [`bezier_eval()`](https://reetamm.github.io/BezELS/reference/bezier_eval.md)
  : Find a point on a 2D Bezier curve B(t) at a particular value of t

- [`bezier_intersection()`](https://reetamm.github.io/BezELS/reference/bezier_intersection.md)
  : Find the intersection of a 2D Bezier curve B(t) and a line passing
  through the origin with slope m

- [`eta_analytical()`](https://reetamm.github.io/BezELS/reference/eta_analytical.md)
  : Evaluate the analytical value of \\\eta\\ for bivariate Gaussian,
  logistic, asymmetric logistic, and inverted logistic copulas

- [`eta_empirical()`](https://reetamm.github.io/BezELS/reference/eta_empirical.md)
  :

  Compute \\\eta\\ based on Bezier spline control points, usually a
  posterior draw obtained from `fit_mcmc_bezier`

- [`fit_mcmc_bezier()`](https://reetamm.github.io/BezELS/reference/fit_mcmc_bezier.md)
  : Main MCMC fit function

- [`gen_data_exp()`](https://reetamm.github.io/BezELS/reference/gen_data_exp.md)
  : Generate data with unit exponential margins from 4 different
  bivariate copulas

- [`get_r0_marg()`](https://reetamm.github.io/BezELS/reference/get_r0_marg.md)
  : Truncated Gamma calculations

- [`lambda_analytical()`](https://reetamm.github.io/BezELS/reference/lambda_analytical.md)
  : Evaluate the analytical value of \\\lambda(\omega)\\ for bivariate
  Gaussian, logistic, asymmetric logistic, and inverted logistic copulas

- [`lambda_empirical()`](https://reetamm.github.io/BezELS/reference/lambda_empirical.md)
  :

  Compute \\\lambda(\omega)\\ based on Bezier spline control points,
  usually a posterior draw obtained from `fit_mcmc_bezier`

- [`plot_bezier_euclidean()`](https://reetamm.github.io/BezELS/reference/plot_bezier_euclidean.md)
  : Plot the limit set based on MCMC output in Euclidean coordinates and
  export median curve

- [`plot_bezier_polar()`](https://reetamm.github.io/BezELS/reference/plot_bezier_polar.md)
  : Plot the limit set based on MCMC output in polar coordinates and
  export median curve

- [`tau_analytical()`](https://reetamm.github.io/BezELS/reference/tau_analytical.md)
  : Evaluate the analytical value of \\\tau_1(\delta)\\ for bivariate
  Gaussian, logistic, asymmetric logistic, and inverted logistic copulas

- [`tau_empirical()`](https://reetamm.github.io/BezELS/reference/tau_empirical.md)
  :

  Compute \\\tau_1(\delta)\\ based on Bezier spline control points,
  usually a posterior draw obtained from `fit_mcmc_bezier`

# Compute \\\eta\\ based on Bezier spline control points, usually a posterior draw obtained from `fit_mcmc_bezier`

Compute \\\eta\\ based on Bezier spline control points, usually a
posterior draw obtained from `fit_mcmc_bezier`

## Usage

``` r
eta_empirical(control_points)
```

## Arguments

- control_points:

  A 9x1 vector of Bezier spline control points

## Value

scalar value of \\\eta\\
